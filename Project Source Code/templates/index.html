Handwritten Digits Recognition

The fundamental problem with handwritten digit recognition is that handwritten digits are not same since they vary from person to person. The project presents recognizing the handwritten digits(0 to 9) from the famous MNIST dataset. Here we will be using artificial neural networks/convolution neural network.
